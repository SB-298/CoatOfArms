# **Coat of Arms** - *A faction icon editor.*

Design a custom icon and use it as the faction icon on the world map.
Open the editor from the world map (select a colony or settlement). Icons are applied per save.
You can assign custom icons to other factions from their settlements (god mode or a setting).
Works with or without Ideology; more emblem categories with DLC.

---

## Features

**Create custom faction icons**
Build an icon in the editor and set it as the faction icon on the world map.
Assign custom icons to non-player factions via their settlements (god mode by default; optional setting to allow anytime).

**Editor**
- **Frames** - Frame styles (e.g. square, rounded) applied around the icon.
- **Background patterns** - Patterns (solid, divided, heraldic) with up to three colours.
- **Borders** - Optional outline with adjustable thickness. Suggested to use less than or equal to 1 thickness.
- **Emblems** - Categorised grid; add multiple layers and stack or overlap symbols.
- **Colour** - Full picker for background and each emblem, including alpha.
- **Per-emblem editing** - Position (X, Y), rotation, scale (linked or unlinked), opacity, flip. Real-time preview.

**Emblem library and DLC**
- Categories: shapes, symbols, lettering, vanilla-style faction icons. Ideology adds more (roles, precepts, structures). Uses vanilla and DLC art when installed.

**Sharing and presets**
- Save and load presets.
- Copy to clipboard to share; others can import from clipboard.
- Export as PNG (mod Exports folder or custom path in settings).

Revert any faction to its default icon from the editor. Reverting a non-player faction after applying a custom icon can sometimes leave the icon with wrong colours (known quirk).

## Requirements

- **Harmony** (required; subscribe via the mod's dependency link)

## Compatibility

Works with base game and all DLC. No DLC required. More emblems with Royalty, Ideology, or Biotech.
Mods that change faction icons may conflict; adjust load order or disable their icon changes if needed.

## FAQ

**Q: Is this compatible with existing saves?**  
A: Yes. It should work with existing saves.

**Q: Can I remove this mid save?**  
A: From what I've tested, yes. It may show an error in the logs.

**Q: Can I add my own emblems, frames and patterns?**  
A: Yes. You can add defs and textures to this mod's folders, or create your own mod with a Defs folder and add defs (EmblemDef, FrameDef, BackgroundPatternDef).
Load your addon after Coat of Arms.

If you want to provide me with new images, I can update the mod to include them. Images should be 512x512. You will be credited.
